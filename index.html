Summary: Latitude vs X



Ths project was to analyse the climate variations with respect to Equator. Sampled with around 100 locations across various latitudes and plotted graphs for each of the variables i.e Temperature, Humidity, Cloudiness and Wind Speed

To accomplish this analysis we pulled data from OpenWeatherMapAPI for over 500 random cities. Then use matplotlib to plot varioues waether aspects vs latitudes. This site provides the visualization of all the data collected and analyzed. You can go to individual pages to view the Plots or else can go the comparison page to see all the graphs and will be able to compare the same. Data page takes us to the Data that was used to perform this analysis
